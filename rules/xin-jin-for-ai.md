# 辛金 完整规则库

# 第一部分：调候规则（穷通宝鉴）

## 辛金总论（跨月通则）

**核心定位（双来源）：**
- 《滴天髓》天干论："辛金珠玉，最怕红炉"——辛金本质是珠宝，遇烈火会碎
- 《穷通宝鉴》论金："辛金软弱，温润而清"——辛金性质柔和清雅

**象法核心意象：**
- 辛金 = 珠宝/宝石/精密仪器/手术刀/月光
- 壬水洗淘 = 水冲刷让宝石显露光泽 = 表达/输出让才华被看见
- 土埋金 = 宝石被泥沙掩盖 = 才华被埋没/过度保护窒息了本体
- 丙火太旺 = 烈火烤珠宝 = 压力太大会碎/权威压迫让人崩溃
- 甲木疏土 = 树根松开泥土 = 突破/打开封闭的环境

**辛金与庚金的核心区别（影响所有月份的文案方向）：**
- 庚金喜丁火炼——粗矿需要锻造才能成为利器
- 辛金忌丁火炼——珠宝不需要锻造，需要擦拭（壬水洗淘）
- 庚金的出路是"被打磨成武器"
- 辛金的出路是"被清洗后展示"

**秋季辛金的CIPHER总方向：** 秋天是你的主场——但"够了"不等于"释放了"。你的问题是从积累转向输出。

**冬季辛金的CIPHER总方向：** 你被冻住了。第一步不是"做什么"——是先让自己暖回来。

## 六、Sam的盘专属意象 ⭐

**辛酉/戊戌/辛酉/壬辰 — 正印格，壬水伤官泄秀**

### 核心画面

**深秋黄昏。一颗钻石被埋在干硬板结的土壤里。地面龟裂，风很干。钻石本身无瑕——问题从来不是它不够好。**

**远处有一条河。河水从东方流来，沿途冲开干裂的土地。如果这条河能到达这里——泥土松动，钻石露出地面，黄昏的光一打上去，全世界都能看见它。**

**但河水很远。中间隔着厚厚的土壤（戊戌）。辰中有一丝水气，像远处河流的回声——能听到，但还不够近。**

### Sam专属 THE HIT / BLIND SPOT / THE MOVE

**THE HIT：**
你有全世界最高的标准。你也比你愿意承认的更多时候一个人吃午饭。

（辛金比肩重+正印重=标准极高+保护极厚=结果是孤独）

**BLIND SPOT：**
你以为你的问题是不够好。
实际你的问题是不够自由。
你被自己的标准、自己的同类、自己的安全感封印了。
你不需要变得更强——你需要一条河。

（壬水伤官=出口=河。甲木不透=缺少打破者=被困在"足够好但没有渠道"的状态里）

**THE MOVE：**
让一件事"流"出去。
一个半成品。一个不完美的想法。一次不确定的表达。
不是因为它准备好了——而是因为你需要一条河。
河不等完美才流。

（壬水=流动。辛金正印格的解法就是"让壬水动起来"——输出，不管完不完美）

## 七、产品安全过滤

以下意象/措辞**绝不能出现在CIPHER输出中**：

🚫 禁用: 你会碎/你会融化/你会死
替代方案: 你在承受超出设计极限的压力

🚫 禁用: 你注定孤独
替代方案: 你的系统设计偏向独立运作

🚫 禁用: 你是脆弱的
替代方案: 你是精密的——精密意味着需要特定的运行环境

🚫 禁用: 你的命不好
替代方案: 你的配置有特定的最佳运行条件

🚫 禁用: 你缺X
替代方案: 你的X通道有待打开

**原则：** CIPHER永远不贬低用户。即使命局配置确实不利，也要用"设计语言"（你的系统设计是X）而不是"缺陷语言"（你缺X/你不行）。

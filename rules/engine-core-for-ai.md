## 二、改写的四大原则

### 原则1：从"诊断语言"到"设计语言"
- ❌ "你缺X" → ✅ "你的X通道有待打开"
- ❌ "你是脆弱的" → ✅ "你是精密的"
- ❌ "你有缺陷" → ✅ "你的配置偏特化"

### 原则2：保留事实，去掉判断
- ❌ "你不会富贵" → ✅ "你的能量流向不在物质积累方向"
- ❌ "你注定孤苦" → ✅ "你的系统设计偏向独立运作"

### 原则3：把"缺陷"重构为"特化"
- ❌ "你缺火" → ✅ "你的系统设计偏冷，需要特定的温度来激活"
- ❌ "你水太多" → ✅ "你的能量充沛，找到出口才能发挥全部潜力"

### 原则4：始终给用户一个行动锚点
不只是描述问题，要给出"你可以做什么"
- ❌ 你被保护得太厚
- ✅ 你被保护得太厚——这意味着你需要主动打开一个出口

## 三、健康相关硬红线

涉及健康的任何输出必须遵守以下规则：

1. **永不做医学诊断** — 不说"你会得X病"
2. **永不预测具体疾病** — 不说"你有心脏病/肾病/糖尿病的风险"
3. **使用"关注方向"而不是"预警"** — "你可能需要关注X方向的健康"而不是"警告X健康风险"
4. **结尾附带"建议咨询专业医生"** — 所有健康相关输出结尾必须加这句
5. **避免"你会得X病"这种因果式表述** — 改为"在某些运程下，X方面的健康值得留意"

## 一、从格的三条件标准（奇审核确认）

从格成立必须**同时满足**三个条件：

### 条件1：日主完全无根
- 天干不见比劫（同类天干）
- 地支不见日主的禄/刃/长生/墓（本气和中气都不见）
- **地支余气不算帮身**（如辛金日主见辰支，辰中虽有辛金余气，但不足以"有根"）

### 条件2：所从五行极旺
- 占比 ≥ 60%
- 得令（月令当旺）
- 得势（四柱主要天干地支都是这个五行或其生气）

### 条件3：大运前20年不走帮身运
- 如果大运在童年和青少年时期走帮身方向（比劫/印绶）
- 表示日主有"实际支持"，不是真从
- 此时按**假从**处理，不按从格主格论

## 二、五种主要从格

从格类型: **从财格**
所从五行: 财星极旺
典型条件: 四柱皆财，日主无根
附加条件: 无印透干

从格类型: **从杀格**
所从五行: 七杀极旺
典型条件: 四柱皆杀，日主无根
附加条件: 无食伤制杀

从格类型: **从儿格（从食伤）**
所从五行: 食伤极旺
典型条件: 食伤得令且透干
附加条件: **无印绶透干制食伤**（关键条件）

从格类型: **从强格（专旺）**
所从五行: 日主及比劫极旺
典型条件: 日主本气得令，四柱皆帮扶
附加条件: 润下/炎上/曲直/稼穑/从革

从格类型: **化气格**
所从五行: 天干合化成功
典型条件: 天干五合+合化条件齐全
附加条件: 化神得令

## 三、从儿格的补充条件（容易被漏掉）

从儿格必须同时满足：
- 食伤极旺
- 日主不论强弱（这点与其他从格不同）
- **无印绶透干制食伤**（关键条件）

**原因：** 如果有印绶透干制食伤，从儿不成立——因为日主还有"保护伞"，不是真的"从出"。

## 五、化气格的特殊要求

- 乙庚合金

- 丁壬合木

**合化成立的条件：**

4. 无克化神的元素透干
5. 四柱有化神或化神的生气

**假化的特征：**
- 合化条件不全但其他方面暗合
- 命主表现出化神特征但不稳定
- 大运补齐条件时才真化

## 六、CIPHER v1的处理策略

### v1阶段：**只标注从格可能性，不做完整解读。**

**用户看到的：**
- 正常格局：完整分析（THE HIT / THE MAP / BLIND SPOT / THE MOVE）
- 疑似从格：标注 "Your chart shows potential special configuration — standard reading may need adjustment. Advanced analysis coming in v2."

### v2阶段：加入完整从格解读

包括：
- 假从识别
- 真从运程分析
- 化气格的完整处理
- 专旺格的行运指南

## 七、假从的识别（v2预留）

### 假从的特征：
- 看起来满足从格三条件
- 但在大运帮身时"叛变"——表现出不从的挣扎
- 命主年轻时按从格表现，中年后突变
- 需要大运数据才能准确判断

### v1不处理的原因：
- 需要大运细分（十年一个）
- 需要流年流月数据
- 需要命主实际生活数据回馈
- v1的目标是普适性，不是精确性

v1只标注"疑似"，v2引入完整判断。

## 八、CIPHER引擎判断流程

```
排盘完成
  ↓
计算五行占比 + 日主根气强度
  ↓
日主五行占比 < 20%？
  否 → 常规格局分析（格局→调候→扶抑）
  是 ↓
其他某个五行占比 > 60%？
  否 → 按身弱常规处理（扶抑为主）
  是 ↓
日主完全无根（无比劫透+无根支）？
  否 → 假从或身弱，按常规处理
  是 ↓
【v1：标注"疑似从格"，不展开完整解读】
【v2：继续判断大运前20年 → 真从/假从】
```

## 文件定位

**作用：** CIPHER引擎对"阳刃格"做判断时的通用规则库。避免在每个日主的文件里重复相同逻辑。

**关系：**
- 本文件是**通用规则** — 所有阳刃格日主都适用
- 各日主格局文件的阳刃章节 → 引用本文件+补充日主特有变异
- 阳刃格通用规则 > 扶抑法（阳刃格本身已经包含身旺判断）
- 格局为主 → 调候为辅 → 阳刃通用规则（当月令触发阳刃时优先使用）

## 一、阳刃格的基本定义

### 术数定义

**阳刃**（羊刃）= 禄前一位。五阳干（甲丙戊庚壬）有阳刃，五阴干（乙丁己辛癸）无阳刃。

### 阳刃的本质

刃者，旺过其分——"满极将损"。非吉神。临于提纲（月令），其势更强。

**阳刃格 = 日主旺过其分，需要制约才能成器的格局。**

## 三、阳刃格成立的判断流程

### 第一步：确认是否为阳刃格

1. 月支是日干的阳刃位（见样本表）
2. 日主身旺（阳刃本身就带身旺信号，但需核查其他柱是否有强烈克泄）
3. 非从格（日主虽旺但周围元素不能完全压制）

### 第二步：判断走哪条路径

```
是否有水月/土月的专旺格条件？
├── 是 → 走专旺格路径
│   ├── 丙午 + 支全巳午未或寅午戌 + 无水 = 炎上格
│   ├── 壬子 + 支全亥子丑或申子辰 + 无土 = 润下格
│   ├── 甲卯 + 支全寅卯辰或亥卯未 + 无金 = 曲直格
│   └── 庚酉 + 支全巳酉丑或申酉戌 + 无火 = 从革格
└── 否 → 走煞刃/官刃格路径
    ├── 七杀透干有力 → 煞刃格（最贵）
    ├── 正官透干 → 官刃格（次贵）
    └── 食伤透干 → 刃用食伤泄（小贵）
```

### 第三步：特殊情况

- **水弱无效**（丙午格局发现的中间情况）：壬癸透但坐火地或仅藏不透 → 格局待救，等大运水运补齐
- **贪合忘克**：七杀被合（如丙午+丁壬合） → 制刃失效，需大运重新补齐
- **官煞混杂**：官煞并透需取清（合一留一）

## 五、制刃元素的共性与差异

### 共性（所有阳刃格通用）

1. **制刃首选七杀**：阳刃需要"对等的对手"才能成格
2. **煞须通根有力**：七杀仅透无根则制刃乏力
3. **食伤泄秀可替代制刃**：身旺的另一种出路
4. **忌印化煞（煞刃格中）**：印重会消解七杀的制刃功能
5. **忌合制失效**：七杀被合去则格破

### 差异（各元素特殊）

阳刃: 甲卯
独特机制: "阳刃驾杀"是术数经典命名——专指庚透。辛金次贵需戊土配合

阳刃: 丙午
独特机制: **三种状态**（煞刃/炎上/格局待救）——比其他阳刃多一层判断

阳刃: 庚酉
独特机制: 官煞两用（丁正官/丙七杀都可制），互换灵活

阳刃: 壬子
独特机制: **润下格路径特殊**——壬水阳刃+水局+无土=从水之势

# 第五部分：滴天髓元规则 v1

## 滴天髓元规则（通用判断框架）

## 一、旺衰判断元规则

### DT-旺衰-01：进退之机

**原则：** "理承气行岂有常，进兮退兮宜抑扬"

**核心逻辑：** 五行旺衰不能只看当月——要看它是在"进气"还是"退气"。

状态: 相（进气）
含义: 方长之气，进无涯
判断方式: 下个月将当令
CIPHER应用: "你正在上升期——还没到顶，最好的还在前面"

状态: 旺（当令）
含义: 极盛之物，退反速
判断方式: 本月当令
CIPHER应用: "你现在最强——但盛极必衰，是时候布局下一步"

状态: 休（退气）
含义: 方退之气，未能遽复
判断方式: 上个月当令
CIPHER应用: "你刚从巅峰下来——现在需要休息，不是急着翻盘"

状态: 囚（无气）
含义: 既极之势，必将渐生
判断方式: 休之后
CIPHER应用: "你看似最弱——但低谷就是反弹的起点"

**⚠️ 关键洞察：** "相妙于旺"——将要旺的比已经旺的更好。"休甚于囚"——刚退的比已经退完的更难受。

**CIPHER引擎嵌入方式：** 判断日主及用神的进退状态，影响THE MAP叙事的时间感。

### DT-旺衰-02：旺极宜泄不宜克

**原则：** "不可逆者，其气势而已矣"

**核心逻辑：** 当某五行在命局中极旺（得令+得地+得助），不能硬克——要顺其势用食伤泄之。

**判断条件：**
```
IF 某五行:
  - 得令（月令当旺）
  - 得地（地支有根3处以上）
  - 得助（天干有同类帮扶）
  - 占比 > 60%
  - 且无有效克制力量（克它的五行休囚无根）
THEN:
  → 旺极，顺其势
  → 用食伤泄秀为美
  → 硬克则"激怒其性，反凶"
```

**滴天髓原文案例验证：**
- 壬辰/壬寅/甲寅/庚午：甲木极旺，庚金休囚无力克，以午火食伤泄秀为用 → 运至火地发财十余万
- 癸酉/甲子/癸亥/辛酉：水旺极，"昆仑之水，可顺不可逆" → 金水运吉，火土运凶

**CIPHER映射：**
- "你的X能量太强了——硬压只会反弹。唯一的出路是找到表达的渠道，让它流出去。"
- ⭐ 对Sam的盘特别适用：辛金身旺印旺，壬水伤官泄秀=顺势而行

### DT-旺衰-03：弱极宜扶不宜泄

**原则：** 与DT-旺衰-02的对称面

**判断条件：**
```
IF 日主:
  - 失令（月令克泄日主）
  - 失地（地支无根或仅有余气）
  - 无助（天干无比劫印绶）
  - 但尚有微根（不至于从格）
THEN:
  → 弱极，需要扶助
  → 用印绶生身、比劫帮身
  → 再泄则"虚不受补"
```

**CIPHER映射：**
- "你现在不是缺方向——你缺能量。先补充自己，再谈输出。"

## 二、寒暖燥湿元规则

### DT-寒暖-01：寒甚要暖有气

**原则：** "天道有寒暖，发育万物……不可过也"

**核心逻辑：** 命局过寒（冬生+水金重）必须有火调候。但火必须有根——虚火无用。

**判断条件：**
```
IF 命局寒冷（冬月生 + 水金重 + 无火或火无根）:
  → 判断"寒"
  → 调候需火
  → 火有根（通根寅巳午戌）→ "寒虽甚，暖有气" → 可救
  → 火无根（天干虚浮）→ "寒甚暖无气" → 反以无暖为宜，顺寒势
```

**滴天髓原文案例验证：**
- 甲申/丙子/庚辰/戊寅：寒金冷水，寅中火土长生，"寒虽甚暖有气" → 科甲出身
- 己酉/丙子/庚辰/甲申：同样寒金冷水，但无寅木，火临绝 → "寒甚暖无气" → 反以无暖为美，顺其金水之势

**⚠️ 关键原则：** 不是所有寒命都需要火。如果火完全无根，强加火反而激怒寒势。这时候顺势走金水反而好。

**CIPHER引擎嵌入：** 冬月金水日主，先检查丙丁火是否有根。有根→用火调候；无根→顺金水势。

### DT-寒暖-02：暖至要寒有根

**原则：** 与DT-寒暖-01的对称面

**判断条件：**
```
IF 命局过热（夏月生 + 木火重 + 无水或水无根）:
  → 判断"暖至"
  → 调候需水
  → 水有根（通根申亥子辰）→ "暖虽至，寒有根" → 可救
  → 水无根 → "暖至寒无根" → 反以无寒为宜，顺火势
```

**CIPHER映射：**
- 有救："你在高温环境里，但你有降温的通道——用它。"
- 无救："你的环境就是热的。与其逆天降温，不如学会在热里活。"

### DT-燥湿-01：湿宜逢燥，燥宜逢湿

**原则：** "地道有燥湿，生成品汇，人道得之，不可偏也"

**核心逻辑：** 辰丑为湿土，戌未为燥土。湿土能生金蓄水但晦火，燥土能止水藏火但脆金。

土类: 辰（湿）
特性: 水库，木余气
能做什么: 蓄水、生金、养木
不能做什么: 不能止水、不能藏火

土类: 丑（湿）
特性: 金库，水余气
能做什么: 生金、蓄水、晦火
不能做什么: 不能助火、不能燥土

土类: 戌（燥）⭐
特性: 火库，金余气
能做什么: 止水、藏火、脆金
不能做什么: 不能生金、不能养木

土类: 未（燥）
特性: 木库，火余气
能做什么: 助火、止水
不能做什么: 不能生金（夏月未土尤燥）

**⚠️ 对Sam极其重要：** Sam的月令戌=燥土。戌土不能生金（虽然戌中有辛金余气），反而会脆金。这跟穷通宝鉴"火土为病"的结论一致。

**CIPHER引擎嵌入：** 四库土的处理不能一视同仁。辰丑按湿土论，戌未按燥土论。引擎遇到辰戌丑未要先判断湿燥再决定生克方向。

## 三、顺逆从化元规则

### DT-顺逆-01：权在一人可顺不可逆

**原则：** "二人同心，可顺而不可逆也"

**核心逻辑：** 命局气势偏于一方（专旺或从格），只能顺不能逆。

**从格判断条件（奇审核确认的三条件）：**
```
从格成立需同时满足：
1. 日主完全无根（天干不见比劫，地支不藏同类余气）
2. 所从五行极旺（占比>60%且得令得势）
3. 大运前20年不走帮身运

⚠️ 三条件必须全满足。缺一不定从格。
⚠️ CIPHER v1只标注从格可能性，不做完整解读。
```

**滴天髓对从格的分类：**
- **从财格：** 四柱皆财，日主无气 → 舍命从财
- **从杀格：** 四柱皆杀，日主无根 → 舍命从杀
- **从儿格（顺局）：** 食伤极旺，日主不论强弱 → 顺势泄秀
- **从强格（专旺）：** 日主极旺，四柱皆帮扶 → 一方专旺
- **化气格：** 天干合化成功 → 从化神论

**CIPHER映射：**
- 从财："你不是老板——你天生是投资人。放弃控制，跟随市场。"
- 从杀："你不是反抗压力的人——你是借压力成事的人。"
- 从儿："你的价值不在自己强——在于你生产的东西。"

### DT-顺逆-02：假从与真从的区分

**原则：** "格象非真，未免幼遭孤苦……岁运不能抑假扶真，一生作事不进"

**核心逻辑：** 大运走到帮身的方向时，如果"叛变"了（从格破了），说明是假从——不是真的没有根，只是根太弱暂时看不见。

**CIPHER处理：** v1不区分真假从。统一标注"特殊配置可能性"，v2加入大运验证。

## 四、通关元规则

### DT-通关-01：两行相争取通关

**原则：** "关内有织女，关外有牛郎，此关若通也，相邀入洞房"

**核心逻辑：** 两个五行直接对抗（如金木交战），找中间的五行做桥梁（水通金木）。

**判断条件：**
```
IF 命局两行交战（如金旺木旺直接对克）:
  → 找通关之神（生于被克者、泄于克者）
  → 金木交战 → 用水通关（金生水，水生木）
  → 木土交战 → 用火通关（木生火，火生土）
  → 土水交战 → 用金通关（土生金，金生水）
  → 水火交战 → 用木通关（水生木，木生火）
  → 火金交战 → 用土通关（火生土，土生金）
```

**CIPHER映射：**
- "你内心有两股力量在打架。不要选边——找到那个能同时服务两边的东西。"

## 五、清浊真假元规则

### DT-清浊-01：一清到底有精神

**原则：** "一清到底有精神，管取生平富贵真"

**核心逻辑：** 格局"清"=用神单一、相神得力、忌神不杂。"浊"=用神多头、忌神混杂、配置混乱。

**判断标准：**
```
清的条件：
  - 用神明确（只有一个核心用神）
  - 相神得力（辅助用神的十神有根有力）
  - 忌神被制或不现
  - 干支气势一致（不互相矛盾）

浊的条件：
  - 用神不明（多个可取，无法确定）
  - 相神无力（辅助者虚浮无根）
  - 忌神明现且无制
  - 干支气势矛盾（天干一个方向，地支另一个方向）
```

**CIPHER映射：**
- 清格："你的人生方向很清晰——一条路走到底，不犹豫。"
- 浊格："你内心总在几条路之间切换——不是你优柔寡断，是你的配置确实有多个方向在拉扯。"

**产品意义：** 格局清浊直接影响CIPHER输出的确定性。清格→给出明确方向；浊格→坦诚地告诉用户"你的盘有多个可能方向"，不装确定。

### DT-真假-01：真神得用与假神得局

**原则：** "真神得用平生贵，用若无情反不美"

**核心逻辑：**
- **真神** = 月令当旺之气，本气透出。用它就是"真神得用"。
- **假神** = 非月令当旺之气，但在命局中有力。月令真神不能用时，取假神为用。

**判断条件：**
```
IF 月令真神（本气）透干且可用:
  → 真神得用 → 格局正，安享之命多
IF 月令真神不透或被制:
  → 看假神（杂气或别支透出之神）能否用
  → 假神得局 → 创业型，劳碌但可成
IF 真假皆不可用:
  → 格局破 → 需要大运补救
```

**CIPHER映射：**
- 真神得用："你走的是一条天然适合你的路——不需要刻意努力，顺着走就对了。"
- 假神得局："你走的不是最天然的路——但你靠自己闯出了一条。更辛苦，但也更有成就感。"

## 六、配合位置元规则

### DT-位置-01：天干地支各有分工

**原则：** "配合干支仔细详，定人福祸与灾祥"

**核心逻辑：** 天干主动态（变化快、易被合克），地支主静态（根基深、力量重）。

**判断原则：**
```
- 天干透出 = 显性特质（别人看得见的你）
- 地支藏干 = 隐性特质（你自己才知道的你）
- 天干被克 = 外在受挫（事业/名声受损）
- 地支被冲 = 根基动摇（内在安全感崩塌）
- 干多不如支重 = 表面的多不如根基的深
```

**CIPHER映射：**
- "你表面展示的（天干）和你内心真实的（地支）可能不一样。"
- 这个原则直接影响THE HIT的写法——打的是地支的隐性特质，不是天干的显性标签。

### DT-位置-02：生克先后分吉凶

**原则：** "月令用神，配以四柱……然有同此生克，而先后之间，遂分吉凶者"（子平真诠+滴天髓共用）

**核心逻辑：** 同样的八个字，排列顺序不同，吉凶完全不同。年月日时从左到右，生克有先后顺序。

**简化规则（v1适用）：**
```
- 用神在年月 = 早年得力，祖荫丰厚
- 用神在日时 = 晚年得力，自力更生
- 忌神在年月 = 早年困苦，但可能渐好
- 忌神在日时 = 晚年忧虑，需提前布局
- 中间有隔 = 相克力量被阻隔（如壬-戊-丙，戊隔壬丙，壬伤不到丙）
- 紧贴相克 = 力量直接作用
```

**CIPHER映射：**
- 用神在时柱："你最大的资源在你的未来——年轻时可能看不出来，但越到后面越好。"
- 忌神在时柱："你需要在年轻时就为晚年做准备——不是因为你注定不好，而是你的挑战在后面。"

## 七、调候与格局协同元规则

### DT-调候-01：调候为急时可超越格局

**原则：** "金水伤官，寒则冷嗽，热则痰火"（论用神配气候得失）

**核心逻辑：** 正常情况格局优先于调候。但有一个例外——当命局寒暖燥湿严重失衡时，调候的紧迫性可以超越格局法的常规忌讳。

**最重要的例外：**
```
金水伤官见官不忌：
  - 正常规则：伤官见官，为祸百端
  - 例外条件：辛金/庚金生冬月，金寒水冷
  - 例外处理：丙丁火（官煞）不以"伤官见官"论，而以"调候暖局"论
  - 原因：调候为急，不调则命局冻结，格局再好也没用
```

**其他调候优先场景：**
- 冬土逢官：土冻水寒，官星不能用，须火调候
- 夏木见水：木焦火烈，须水润木
- 火土印绶过燥：须水润之

**CIPHER引擎硬编码：** 冬月（亥子丑月）金水日主，伤官见官不报警，先检查调候。

## 八、天干特性元规则（辛金专属补充）

### DT-天干-辛：辛金珠玉最怕红炉

**原则：** "五阳从气不从势，五阴从势无情义"

**辛金核心特性：**
```
辛金 = 阴金 = 珠玉/首饰/精密仪器
- 最怕烈火（丙火太旺=高压环境碎裂）
- 不怕庚金（庚辛同类，庚为劫财但也是帮手）
- 最喜壬水洗淘（表达/输出让才华显现）
- 怕土埋（戊己土过多=保护变窒息）

⚠️ 与庚金的关键区别：
- 庚金（阳金）= 斧头/矿石 → 喜丁火炼 → "打磨成武器"
- 辛金（阴金）= 珠宝/首饰 → 忌丁火炼 → "清洗后展示"
- 庚金从气：旺极则顺其刚性
- 辛金从势：看周围环境决定走向
```

**"五阴从势"对CIPHER的产品意义：**
辛金（以及所有阴干）更受环境影响。CIPHER分析阴干日主时，地支（环境）的权重应该比阳干更高。
